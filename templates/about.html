About This Application

File Upload and Analysis Tool

Purpose

This application allows users to upload files and automatically analyzes them to provide useful information about the file's content, structure, and properties.

Features

  • File Type Detection: Automatically identifies file types and MIME types
  • Size Analysis: Shows file size in human-readable format
  • Text File Analysis: Counts lines, words, and characters
  • CSV File Analysis: Identifies rows and columns
  • JSON File Analysis: Validates JSON structure and shows metadata
  • Content Preview: Displays preview of file content when applicable
  • Secure Upload: Validates file types and sanitizes filenames

Supported File Types

Text files: .txt
Documents: .pdf, .doc, .docx
Images: .png, .jpg, .jpeg, .gif
Data files: .csv, .json, .xml

Technical Details

  • Built with Python Flask framework
  • Maximum file size: 16 MB
  • Secure filename handling with Werkzeug
  • Responsive web interface
